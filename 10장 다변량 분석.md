# 10장 다변량 분석
* 주성분분석(PCA, principal component analysis)
    * 많은 변수에 포함되어 있는 정보를 적은 수의 변수로 나타내고 싶을때 사용하는 방법
    * 독립변수가 k개 있을 때, k개보다 적거나 같은 서로 직교하는 변수로 나타내기 위한 방법
    * 주성분
        * 데이터의 분산이 최대가 되는 방향으로 생성된 변수(VARIABLE)
    * 주성분 스코어(= 점수, 득점)
        * 개개의 데이터에 대해  계산되는 각 주성분의 값(VALUE)
    * 공분산 행렬에서 고유값 분해를 통해 진행함
        * 변수 간에 스케일이나 분산이 크게 다를 때는 상관행렬을 이용함
        * 고유값 : 주성분 스코어의 분산, 정보량을 나타냄
            * 고유값이 큰 것부터 제 1, 제 2, ... 주성분이라고 함
        * 고유벡터 : 주성분의 계수
            * 각 주성분계수는 계수 "제곱합 = 1" 를 제약조건으로 한 최적화문제(분산최대화)의 해답
                * 라그랑수 승수법이 활용됨
    * 기여율
        * 각 주성분에 의해 원 데이터 정보의 몇 퍼센트가 설명될지를 나타냄
        * 각 주성분에 의해 집약되는 정보의 비율
        * 기여율 = 각 고유값 / 고유값의 총합
    * 누적기여율
        * 기여율을 제 1주성분부터 순서대로 누적한 것
        * 보통 0.8(80%)를 목표
    * 주성분부하량(인자부하량)
        * 주성분과 원 변수의 상관정도를 나타냄
        * sqrt(고유값) * 고유벡터로 계산됨
        * 인자부하량의 크기와 부호를 보면서 주성분을 해석함
* 클러스터 분석
    * 개체 간의 유사성 측정
        * 대표적으로 유클리드 거리를 활용함
    * 개체를 클러스터에 결합시키는 방법
        * 무게중심법
            * 개체와 각 클러스터의 중심 간에 거리를 구하고, 가까운 클러스터에 결합함
        * 워드법
            * 개체를 각 클러스터에 결합시킬 경우의 변동(편차제곱합)을 구해서 변동의 증가량이 최소가 되는 클러스터 결합
    * 계층 클러스터
        * 계층 구조(시각화)로 개체를 분류(군집화)하는 방법으로, 개체의 수가 적을 때 적합함
        * 클러스터링은 수형도를 그리고, 수형도를 수평 방향으로 자르는 절단을 통해 이루어짐
            * 수형도
                * 클러스터나 개체의 결합과정을 나타낸 나뭇가지 모양의 그림
                * 가로축에 개체, 세로축에 결합시의 거리(비유사성)을 나타냄
                    * 거리가 짧도록 결합을 진행하는 것이 바람직
            * 수형도의 절단은 수직 방향의 선분 길이(거리)가 긴 곳에서 진행함
            * 연쇄사슬
                * 수형도가 기존의 클러스터에 하나씩 개체가 결합되는 형태를 띄고 있음
                    * 제대로 분류되어 있지 않은 분석결과의 전형적인 형태
    * 비계층 클러스터
        * 계층 구조를 만들지 않고 개체의 분류만 하는 방법으로, 개체의 수가 많을 떄 적합함
        * 대표적인 알고리즘으로 K-means가 있음
* **필요시 추가적으로 공부 필요한 목록**
    * 인자분석(factor analysis)
        * 여러 변수의 배후에  공통으로 존재하는 공통인자를 추출해 변수 간의 관련성을 이해함
            * 공통인자
                * 관측된 변수들에 공통으로 포함되어 있는 인자
                * 변수간 상관관계를 생성시키는 "관측불가"한 인자
                * 2개 이상의 변수에 영향을 주는 인자
                * 인자모델 : xi(관측변수) = ai(인자부하량) * F(공통인자) + ui(유일인자)
                    * 인자부하량을 구하는 것이 인자분석의 목적
                        * 최대우도법이나 주성분법이 많이 이용됨
                        * 인자부하량의 제곱합은 공통성이라고 함
                            * 공통성
                                * 각 변수가 갖고 있는 정보가 인자 모델에 반영되어 있는지를 나타냄
                                * 공통성이 작은 변수는 모델에서 삭제하고 재추정하는 것이 좋음
                                    * 보통 공통성이 0.3보다 작을 경우에 해당함
                        * 회전
                            * 인자부하량 값에 눈에 띄는 경향이 없는 경우 해석하기 쉽게 인자 축을 회전시키는 것
                            * 공통인자 간에 상관을 가정할 수 없는 경우에는 직교회전, 상관을 가정할 수 있는 경우에는 사교회전을 함
                                * 직교회전
                                    * 축은 직교를 유지한채 회전
                                * 사교회전
                                    * 축이 따로따로 회전
                * 공통인자의 수는 사전의 공통인자의 수가 상정 가능한 경우를 제외하고, 데이터의 고유값을 계산하고 1 이상인 고유값의 수를 채용함
        * 소비자 의식이나 브랜드 이미지, 가치관 분석 등에 이용하는 경우가 많음
        * 바이플롯(biplot)
            * 인자부하 플롯과 인자점수 플롯을 하나로 그린 것
    * 구조 방정식 모델링(SEM)
    * 대응분석
